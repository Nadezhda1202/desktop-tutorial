# Авторизация пользователя в Sherpa AI Server

Для осуществления локальной авторизации пользователя необходимо ввести логин и пароль учетной записи на отдельной странице авторизации Sherpa AI Server. Помимо локальной авторизации пользователя Sherpa AI Server поддерживает также доменную авторизацию (через Active Directory, протоколы LDAP/LDAPS) и OpenID-авторизацию.

Время жизни сессии без активных действий пользователя — 15 минут.&#x20;

Кнопка Logout очищает текущую сессию пользователя и возвращает пользователя на экран авторизации. Один пользователь может быть участником только одной учетной записи (аккаунта).
