# Версия 141857

**Вышел новый релиз Sherpa Orchestrator: Версия 141857.**

### Новая функциональность

* **Фильтрация задач по периоду** — теперь удобно работать с большим количеством задач без ограничений.
* **Приоритет запуска триггеров** — триггеры запускаются в порядке приоритета, что повышает контроль за процессами.
* **Проверка обновлений** — кнопка в интерфейсе позволяет сразу узнать текущую версию и перейти на страницу релизов.
* **Расширенный поиск ресурсов** — при поиске по имени теперь отображаются все ресурсы с одинаковым именем.
* **Оптимизация загрузки задач** — работа с большим количеством сообщений роботов стала быстрее.
* **JSON-формат данных** — все запросы на получение задач возвращают структурированные данные в формате JSON.
* **Тёмная тема** — новый режим интерфейса с возможностью переключения между светлой и тёмной темами.
* **Подключение по VNC** — теперь можно подключаться к роботу напрямую из интерфейса.

### UI/UX улучшения

* Улучшена работа с таблицами данных, формами и страницей логина.
* Переименована вкладка «Учетная запись» в меню API на «Аккаунт» для единообразия интерфейса.
* Общие визуальные улучшения для более удобного и современного интерфейса.
