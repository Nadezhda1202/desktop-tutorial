# Вкладка "Прокси"

Вкладка “Прокси” в окне настроек Sherpa Assistant позволяет управлять настройками прокси. Можно использовать системные настройки прокси, а так выбрать режим “Без прокси” или настроить прокси вручную.&#x20;
