# Цикл Для каждого - Список (ForeachList)

Перебирает все элементы в заданном Списке, в каждой итерации цикла присваивает текущий элемент переменной, которая указана в свойстве "Элемент", а затем выполнение сценария продолжается в сторону выхода "Элемент" данного блока (в так называемое "тело цикла").&#x20;

<figure><img src="../../../.gitbook/assets/блок цикл для каждого.png" alt=""><figcaption></figcaption></figure>

После перебора всех элементов списка выполнение сценария продолжается в сторону нижнего выхода блока. Не забудьте соединить выход последнего блока в теле цикла со входом данного блока.
