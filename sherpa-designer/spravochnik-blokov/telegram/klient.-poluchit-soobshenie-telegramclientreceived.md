# Клиент. Получить сообщение (TelegramClientReceived)

Получает сообщение из очереди. Если сообщений в очереди нет, то генерируется ошибка.

Создать подключение" из группы Telegram, который ранее использовался для подключения клиента Telegram.</td></tr><tr><td valign="top">Ждать сообщение</td><td valign="top">При включении блок будет ждать новые сообщения.
