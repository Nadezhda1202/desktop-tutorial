# Удалить строку/колонку/ячейку (DeleteExcelRange)

Удаляет строку, колонку или ячейку из указанной ячейки в Excel-документе.

Название переменной в данном свойстве должно совпадать с названием переменной в свойстве "Ссылка на документ", блоков "Создать документ" или "Открыть документ" из группы Excel, которые ранее использовались для создания или открытия данного документа.</td></tr><tr><td>Лист</td><td>[Число/Текст] ID (порядковый номер) или название листа, на котором расположена требуемая ячейка. Нумерация начинается<br>с 1.</td></tr><tr><td>Строка</td><td>[Число] Номер строки, на которой расположена требуемая ячейка. Нумерация начинается с 1.</td></tr><tr><td>Столбец</td><td>[Число] Номер столбца, в котором расположена требуемая ячейка. Нумерация начинается с 1.</td></tr><tr><td>Имя ячейки</td><td>[Текст] Имя ячейки. Если значение задано, то игнорируются свойства "Строка" и "Столбец". Например: "B4".</td></tr><tr><td>Удалить</td><td><p>Удаляемое значение. Возможные варианты: </p><ul><li>Row - удалить строку; </li><li>Column - удалить колонку; </li><li>CellUp - удалить ячейку со сдвигом вверх; </li><li>CellLeft - удалить ячейку со сдвигом влево.</li></ul></td></tr><tr><td>Количество</td><td>[Число] Количество удаляемых значений.</td></tr><tr><td>Уровень обработки</td><td><p>Выбор уровня обработки ошибок.
