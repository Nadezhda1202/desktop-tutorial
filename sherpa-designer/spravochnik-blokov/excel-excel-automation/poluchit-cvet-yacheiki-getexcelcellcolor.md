# Получить цвет ячейки (GetExcelCellColor)

Получает цвет указанной ячейки.

Нумерация начинается с 1.</td></tr><tr><td>Строка</td><td>[Число] Номер строки, на которой расположена требуемая ячейка. Нумерация начинается с 1.</td></tr><tr><td>Столбец</td><td>[Число] Номер столбца, в котором расположена требуемая ячейка. Нумерация начинается с 1.</td></tr><tr><td>Имя ячейки</td><td>[Текст] Имя ячейки. Если значение задано, то игнорируются свойства "Строка" и "Столбец". Например: "B4".</td></tr><tr><td>Цвет текста</td><td>[Текст] Возвращается цвет текста. Значение имеет формат "#RRGGBB".
