# Запустить код JavaScript (RunJavaScript)

Запускает код JavaScript в веб-странице на указанной вкладке браузера.

При этом произойдет переподключение к текущей вкладке и будет выдержана пауза в 3 секунды перед каждой попыткой.</td></tr><tr><td valign="top">Результат</td><td valign="top">[Строка] Результат выполнения кода на странице.
