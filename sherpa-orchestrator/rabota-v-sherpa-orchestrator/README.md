# Работа в Sherpa Orchestrator

Пользовательский интерфейс Sherpa Orchestrator, открываемый по умолчанию, состоит из основных областей:

1. Панель “Параметры пользователя”.
2. Рабочая область.
3. Панель “Экраны”.
