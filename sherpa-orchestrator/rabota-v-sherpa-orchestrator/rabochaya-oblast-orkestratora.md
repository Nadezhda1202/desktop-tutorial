## **Действия с данными таблиц экранов Оркестратора**

Действия с данными таблиц одинаковые для всех экранов:

* изменение порядка колонок;
* сортировка;
* настройка отображения колонок;
* фильтр;
* настройка видимости колонок.

Для изменения порядка колонок необходимо зажать левую кнопку мыши на выбранной колонке и перенести ее.

Для выполнения сортировки необходимо нажать левой кнопкой мыши по названию нужной колонки, после чего рядом с названием появится стрелка ![](https://lh7-rt.googleusercontent.com/docsz/AD_4nXfF997SBilzX-SNCjV8vnpqpg-0cNwx8R34MPgfNNv9MjyX-PKdetfUDCZCeDQxXdmIWGDCnmtTIbcFsbCMAJ1_oOCDo8UPEfgV9tU9nZWjTZWJFaQCyySt_-XhC1_eaveMTSTKp4Rg4iqaip0D8IK0MM_v?key=PszdKxucB-Fj9Rb8J6q7Vg), которая покажет сортировку данных. Для текстовых колонок сортировка происходит в алфавитном порядке. Для колонок с типом даты - сортировка происходит по дате.

Чтобы получить доступ к настройкам отображения колонок, необходимо нажать на кнопку ![](https://lh7-rt.googleusercontent.com/docsz/AD_4nXf0m604U-SlPaIPiFqwBr0vj2xSYgGzflGbrlcWZcVJ3UPoP8vuaVbZcOtNOFPokTe4-opQ6e3YrZgKqwQ9l0BSjAF9_Am4dOoiGRurng-XIVhFrweYck-OjjSVWWBlxAaCF6hVLMRHSr8S1qirtNYbJ9ci?key=PszdKxucB-Fj9Rb8J6q7Vg). Данная опция позволяет:

1. Pin column – закрепить выбранную колонку:

* Pin Left – закрепить выбранную колонку слева;
* Pin Right – закрепить выбранную колонку справа;
* No Pin – не закреплять выбранную колонку.

2. Autosize This Column – установить автоматический размер выбранной колонки;
3. Autosize All Column – установить автоматический размер всех колонок таблицы;
4. Reset Columns – очистить настройки отображения выбранной колонки.
