# /api/job/read/{guid}

Чтение Работы по GUID

* Эндпоинт: /api/job/read/{guid}&#x20;
* Метод: GET;
* Авторизация: обязательна;
* Параметры:

Guid (обязательный): GUID Версии Процесса для получения Процесса, для которого создается новая Версия.
