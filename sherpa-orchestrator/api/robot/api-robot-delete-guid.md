# /api/robot/delete/{guid}

Удаление Робота по его GUID

* Эндпоинт: /api/robot/delete/{guid}&#x20;
* Метод: DELETE;
* Авторизация: обязательна;
* Параметры:

guid (обязательно): GUID Робота.

* Примеры:

| Запрос                                                  | Ответ     |
| ------------------------------------------------------- | --------- |
| /api/robot/delete/554ab883-1f82-48e1-bb12-5049002e7d70  |       \[] |
