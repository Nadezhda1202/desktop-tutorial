# /api/log/purge

Удаление Логов, принадлежащих текущей учетной записи и созданных до выбранного времени

* Эндпоинт: /api/log/purge&#x20;
* Метод: DELETE;
* Авторизация: обязательна;
* Параметры:

Time (обязательно): временная метка в формате Unix (Unix timestamp);

* Пример:

| Запрос                                                                           | Ответ  |
| -------------------------------------------------------------------------------- | ------ |
| <p>```json</p><p>   {</p><p>   "Time": "1590481487",</p><p>   }</p><p>   ```</p> |   \[]  |
