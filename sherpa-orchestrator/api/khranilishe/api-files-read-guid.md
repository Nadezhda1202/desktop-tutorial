# /api/files/read/{guid}

Получение информации о файле по GUID

* Эндпоинт: /api/files/read/{guid}&#x20;
* Метод: GET;
* Авторизация: обязательна;
* Параметры:

Name (обязательно): Название нового файла.

* Пример:

| Запрос                                                                           | Ответ  |
| -------------------------------------------------------------------------------- | ------ |
| <p>```json</p><p>   {</p><p>   "Name": "Test Robot",</p><p>   }</p><p>   ```</p> |   \[]  |
