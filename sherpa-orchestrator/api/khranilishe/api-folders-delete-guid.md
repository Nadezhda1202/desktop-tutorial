# /api/folders/delete/{guid}

Удаление папки по GUID

* Эндпоинт: /api/folders/delete/{guid}
* Метод: DELETE;
* Авторизация: обязательна;
* Параметры:

Guid (обязательно): Guid папки;

* Пример:

| Запрос                                                    | Ответ  |
| --------------------------------------------------------- | ------ |
| /api/folders/delete/554ab883-1f82-48e1-bb12-5049002e7d70  |   \[]  |
