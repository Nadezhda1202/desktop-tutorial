# /api/account/read/{guid}

Получение информации об Аккаунте по GUID

* Эндпоинт: /api/account/read/{guid}&#x20;
* Метод: GET;
* Авторизация: обязательна;
* Параметры:

GUID (обязательно): GUID Аккаунта.
